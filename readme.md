# Current Year Shortcode

## Beschreibung

Das "Current Year Shortcode"-Plugin ist ein einfaches WordPress-Plugin, das es ermöglicht, das aktuelle Jahr mithilfe des Shortcodes `[current_year]` anzuzeigen. Dies ist besonders nützlich für dynamische Copyright-Hinweise, sodass das Jahr automatisch aktualisiert wird, ohne dass eine manuelle Änderung erforderlich ist.

## Installation

1. **Herunterladen**: Lade die Plugin-Datei herunter und entpacke sie, falls nötig.
2. **Hochladen**: Lade den Plugin-Ordner in das Verzeichnis `/wp-content/plugins/` auf deinem Server hoch.
3. **Aktivieren**: Gehe in deinem WordPress-Dashboard zu **Plugins** und aktiviere das Plugin **Current Year Shortcode**.

## Verwendung

Um das aktuelle Jahr auf einer Seite oder in einem Beitrag anzuzeigen, füge einfach den folgenden Shortcode hinzu:

```
[current_year]
```

Dieser Shortcode zeigt das aktuelle Jahr im Format `YYYY` an, z. B. `2024`. Dies ist ideal für dynamische Copyright-Hinweise am Ende einer Website.

## Beispiel

```
&copy; [current_year] Dein Unternehmen. Alle Rechte vorbehalten.
```

Durch die Verwendung dieses Shortcodes wird das aktuelle Jahr automatisch eingefügt, was die Wartung von Copyright-Hinweisen vereinfacht.
